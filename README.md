강의 타겟

- 머신 러닝이 궁금한 사람

- 수학적인 지식이나 CS 배경지식이 없거나 약한 사람도 이해할 수 있는 수준

  y = Wx + b 이정도 수학 이해하면 따라올 수 있고 ML 이해 가능

- ML 블랙박스처럼 안에 뭐들었는지 몰라도 입력 출력을 보고 원하는 일을 할 수 있다.

  좀만 더 이해하면 블랙박스 효율적으로 사용 가능
